🪨📄✂️ Jokenpô em Python

Projeto simples de Jokenpô (Pedra, Papel e Tesoura) desenvolvido em Python, com foco no aprendizado de lógica de programação, estruturas condicionais e interação via terminal.

🚀 Funcionalidades
Escolha entre Pedra, Papel ou Tesoura
Jogada do computador gerada automaticamente
Exibição visual das jogadas (ASCII Art)
Sistema de vitória, derrota e empate
Loop contínuo até o usuário sair
🧠 Objetivo

Este projeto foi criado como um dos primeiros exercícios práticos para reforçar conceitos básicos:

if/elif/else
Entrada de dados com input
Uso de dicionários
Geração aleatória com random
▶️ Como executar
python nome_do_arquivo.py
