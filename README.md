🪨📄✂️ Jokenpô em Python

Este é um projeto simples de Jokenpô (Pedra, Papel e Tesoura) desenvolvido em Python, com o objetivo de praticar lógica de programação e interação via terminal. O usuário joga contra o computador, que gera escolhas aleatórias, e o sistema determina automaticamente o resultado da partida.

O jogo apresenta as jogadas de forma visual utilizando ASCII Art, tornando a experiência mais clara e interativa. Além disso, o programa roda em loop, permitindo múltiplas partidas até que o usuário decida encerrar.

Este projeto foi um dos meus primeiros passos na programação, servindo como base para consolidar conceitos fundamentais como estruturas condicionais (if/elif/else), manipulação de entradas com input, uso de dicionários e geração de valores aleatórios com o módulo random.

Para executar o projeto, basta rodar o arquivo Python no terminal:

python nome_do_arquivo.py

Apesar de funcional, o código ainda possui pontos de melhoria, como repetição excessiva de lógica, validação de entrada limitada e a escolha do computador sendo feita fora do loop principal. Esses aspectos podem ser otimizados em versões futuras, junto com a implementação de novas funcionalidades, como um sistema de pontuação.
